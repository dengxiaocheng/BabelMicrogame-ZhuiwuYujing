## Scene Objects

### 1. Tower（塔上部）
- **位置**：画面顶部中央，横跨 Zone A-C
- **作用**：展示前兆动画的舞台
- **视觉**：灰色矩形建筑，前兆动画叠加其上
- **交互**：不可直接操作，纯视觉层

### 2. Work Zones（A/B/C）
- **位置**：画面中下部，三个等宽矩形区域
- **作用**：拖拽预警旗的目标区域
- **视觉**：浅色矩形，内含工人图标（简笔画/circle）
- **状态视觉**：
  - 默认：浅灰底色，工人正常图标
  - 已放旗：底色变黄，显示🚩，工人闪烁（准备撤离）
  - 结算-事故：底色变红，工人消失或显示受伤图标
  - 结算-安全：恢复默认
- **交互**：接收拖拽的预警旗；点击已放置的旗可回收

### 3. Warning Flags（预警旗）
- **位置**：底部 Flag Bar
- **作用**：玩家的操作资源（warning_tokens 的具象化）
- **视觉**：3 面小旗图标，已使用的变灰
- **交互**：
  - 拖拽：从 Flag Bar 拖到 Zone（dragstart → drag → drop）
  - 回收：点击 Zone 内已放置的旗，旗回到 Flag Bar
- **限制**：最多 3 面/波，不可跨波

### 4. HUD
- **位置**：画面顶部横条
- **内容**：wave, quota, injury_risk, false_alarm 的实时数值
- **交互**：不可操作，纯信息展示

## Drag & Drop Flow

```
1. dragstart（Flag Bar 中的旗）
   → 视觉：旗变半透明，生成拖拽幽灵
   → 前置检查：warning_tokens > 0 && 当前相位 == 前兆期

2. drag（拖拽中）
   → 视觉：经过 Zone 时 Zone 高亮边框
   → 无 state 变化

3. drop（放到 Zone 上）
   → 前置检查：Zone 尚未被预警
   → state: warning_tokens--, Zone 标记为已预警
   → 视觉：旗出现在 Zone 内，Zone 底色变黄
   → 反馈：工人闪烁提示

4. 回收（点击 Zone 内的旗）
   → state: warning_tokens++, Zone 取消预警标记
   → 视觉：旗回到 Flag Bar，Zone 恢复默认
```

## Feedback Channels

### 即时反馈（操作级）
- 拖拽旗到 Zone → Zone 高亮 + 旗出现
- 回收旗 → Zone 恢复 + 旗回 Flag Bar
- tokens 用完 → 剩余旗变灰，拖拽锁定

### 结算反馈（波级）
- 事故 Zone → 红色闪烁 + 工人受伤动画
- 安全 Zone → 绿色闪烁 + 工人继续工作动画
- HUD 数值跳动更新（quota/injury_risk/false_alarm）
- 每波结算后显示本波 summary："+4 quota, +25 injury_risk"

### 波次反馈（阶段级）
- 新波开始 → 波次数字更新 + "Wave N 开始"
- 前兆期 → 倒计时显示
- 决策锁定 → "决策锁定！"提示
- 结算期 → 短暂暂停 + 结算动画

## Forbidden UI

- 不允许做动作闪避（不是反应速度游戏）
- 不允许只用"预警/不预警"全局按钮
- 不允许用下拉菜单或文字选项列表替代拖拽
- 不允许用 alert/prompt/confirm

## Acceptance Rule

1. 首屏必须让玩家看到至少一个可直接操作的场景对象（预警旗）
2. 玩家拖拽旗到 Zone 必须产生即时可见反馈（Zone 高亮 + 旗出现）
3. 反馈能追溯到 Required State（tokens 减少、Zone 状态变化）
4. 不得只靠随机事件文本或普通选择按钮完成主循环
