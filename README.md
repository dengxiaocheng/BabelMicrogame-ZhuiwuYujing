# 坠物预警

> 识别塔上坠物前兆，决定何时让工人躲避而不耽误产量

## 在线试玩

https://dengxiaocheng.github.io/BabelMicrogame-ZhuiwuYujing/

## 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或使用本地 HTTP 服务器
npx serve .
```

## 测试

```bash
npm test
```

## 核心循环

观察前兆 → 发出预警或继续工作 → 工人反应 → 事故/产量结算 → 下一波

## 操作说明

1. **观察前兆**：观察三个区域上方的塔结构，识别裂缝、晃动、阴影等危险信号
2. **放置预警**：拖拽底部的预警旗到认为危险的区域（预警次数有限）
3. **确认结算**：点击右下角按钮推进阶段，查看事故/产量结算结果
4. **目标**：在 10 波内平衡产量与安全，避免崩溃

## 技术栈

纯 HTML5 Canvas + 原生 JavaScript，无需构建工具。
